# Entrata Selenium Tests

This project contains Selenium tests for the Entrata website, utilizing the TestNG framework in Java. The tests validate various functionalities of the website, including page titles, mouse hover actions, and form interactions.

## Prerequisites

To run the tests, ensure you have the following installed:

- **Java JDK** (version 8 or higher)
- **Maven** (for managing dependencies)
- **Eclipse IDE** (or any preferred Java IDE)
- **Chrome Browser**
